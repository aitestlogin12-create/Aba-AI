# TimeTrack Pro - Employee Time Tracking App

## Project Overview
A comprehensive employee time tracking application with authentication, dashboard, time entry management, and reporting.

## Tech Stack
- Next.js 14 (App Router), React 18, Tailwind CSS 3
- PostgreSQL with Prisma ORM
- NextAuth.js v4 with credentials provider
- Recharts for data visualization

## Design
- Purple primary accent color (from default theme)
- Sidebar layout (AppShell-style) for authenticated pages
- Auth pages use centered card layout
- Uses project's built-in UI components (Card, Button, Badge, Dialog, etc.)
- Animations via FadeIn, SlideIn, Stagger from animate components

## Pages
1. `/auth` - Login/Signup page
2. `/dashboard` - Overview with clock in/out, stats cards, weekly chart, calendar, recent entries
3. `/time-entry` - CRUD for time entries with search, add/edit dialog, delete confirmation
4. `/reports` - Filterable reports with date ranges, daily bar chart, project pie chart, CSV export

## User Roles
- **ADMIN**: Can view all employees' time, filter by employee
- **EMPLOYEE**: Can only view own time

## Database Schema
- User (id, name, email, password, role [ADMIN/EMPLOYEE])
- TimeEntry (id, userId, date, startTime, endTime, duration, project, task, notes, isClockIn)

## API Routes
- `/api/signup` - User registration
- `/api/auth/login` - Login
- `/api/auth/[...nextauth]` - NextAuth handlers
- `/api/time-entries` - GET (list), POST (create)
- `/api/time-entries/[id]` - PUT (update), DELETE
- `/api/time-entries/clock` - GET (status), POST (clock in), PUT (clock out)
- `/api/reports` - GET (report data with summaries)
- `/api/reports/export` - GET (CSV export)
- `/api/users` - GET (admin only, list users)
